PlantApi

Ofrecemos una plataforma intuitiva donde podrás explorar diferentes tipos de plantas, obtener consejos sobre su cuidado y mantener un registro de las que tienes en casa. Encontrarás información detallada para cada tipo de planta, trucos y recomendaciones para poder mantenerlas saludables y mucho más.

¡Comienza ahora!

GET 127.0.0.1:3000/plantas

Obtener todas las plantas disponibles en la base de datos. usando las query params se puede filtrar por name, limit y sort.

GET / PUT / DELETE 127.0.0.1:3000/plantas/id

Obtener una planta en particular por su id, modificarla o eliminarla.

POST 127.0.0.1:3000/plantas

Agregar una nueva planta a la base de datos, keys: name, tipo y cuidados.

GET 127.0.0.1:3000/cuidados

Obtener todos los cuidados disponibles en la base de datos.

GET / PUT / DELETE 127.0.0.1/cuidados/id

Obtener un cuidado en particular por su id, modificarlo o eliminarlo.

POST 127.0.0.1/cuidados

Agregar un nuevo cuidado a la base de datos, keys: tipo, descripcion y frecuencia.
